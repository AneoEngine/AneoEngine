The AneoEngine Operating System

Debian based console enviornment

The stupid Arch goblins try to persuade Python on new devs. How about we give them NASM and C documentation so they can make their own
compiler... God please stop them!

AneoEngine is a lightweight and stripped down version of Debian 13 Linux. It's an x86-64 UEFI operating system. It's useless utilities 
were removed, and the other utilities were rewritten in either standard C with standard STD libraries, or with the AneoC Compiler. Its
basic enviornment is TMUX, the whole thing is console based in fact. Everything is pure! No internet! No bloatware! No useless
utilities! Everything is rewritten (just about)
